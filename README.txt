INSTALACIÓN DEL PROYECTO

1.- Descargar el proyecto del repositorio
2.- Lanzar el comando composer install para instalar las dependencias del proyecto.
3.- Comprimir la carpeta en .zip e instalarlo en wordpress.

CONFIGURACIÓN DEL PLUGIN EXPERIENCE IT

A.- En el Backoffice de wordpress, ir a Plugins/Añadir Nuevo Plugin y Subir Plugin. Seleccionar el .zip y Activar el Plugin.
B.- Aparecerá un nuevo menú en la barra lateral llamado "Configuración Experience IT Plugin". Pulsar para ir a la página de configuración.
C.- En las preferencias se pueden importar el número de usuarios que selecciones introduciendo un número y pulsando "Importar Usuarios". Igualmente se pueden eliminar los datos de los usuarios.
